hexo博客绑定自己的域名

发表于 2019-03-22

　　很早之前就建立了博客，也提交了百度收录网址，但一直没有回音，后来才发现github禁止了百度爬虫，所以想要让百度能搜索出我的博客，只能购买域名指向github。

1、购买域名

　　要想绑定自己的域名首先得先要有一个自己的域名，购买途径很多，这边就不详细说明了，自己百度一下很多，我这边已经有一个nianzx.com的域名了，这边假设读者已经有一个属于自己的域名。

2、建立CNAME文件

　　第二步需要在本地的hexo项目根目录的source文件夹里创建CNAME文件，, 文件里面写上域名信息，如：nianzx.com。注意不要带任何前缀，如果写的是www.nianzx.com那只能用www.nianzx.com访问。

3、上传hexo

　　执行hexo clean && hexo g && hexo d

4、域名解析

　　第四步我们登录万网，找到域名控制台　点击解析　添加记录，添加如下记录后面的等待dns解析完毕即可用自己域名访问博客。

5、写在最后

　　一般10分钟左右即可完成解析工作。如果绑定了个人域名后访问出现404，可能是域名没有实名认证导致的，可以尝试实名认证。
